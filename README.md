# ✈️ TravelDeal — Агрегатор путешествий

**[🌍 Открыть сайт](https://suschkova-arch.github.io/TravelDeal/)** | **[📧 Написать нам](mailto:MILEDI_HR@mail.ru)**

Агрегатор путешествий с **живыми ценами на отели и авиабилеты** от Aviasales, Hotellook и партнёров Travelpayouts. Экономия до 60% с отелей и билетов в любую точку мира.

## 🌍 Что на сайте

### Для путешественников
- **🏨 Живой поиск отелей** — реальные цены из базы Hotellook (Aviasales Group) в рублях для 45+ городов: Россия (от Калининграда до Сахалина), Турция, Египет, ОАЭ, Таиланд, Мальдивы, Италия, Испания, Кипр, Куба, Филиппины и др.
- **✈️ Авиабилеты** — 13 маршрутов с **LIVE-ценами** из базы Aviasales, обновляются автоматически
- **🚆 Ж/Д билеты** — 9 направлений по России и Европе (Сапсаны, Стрижи, Ласточки) с ценами на плацкарт, купе, люкс
- **🌍 Подборка популярных отелей** — 28 отелей с фильтрами по стране, звёздам и питанию, сравнение цен у 4+ партнёров
- **📈 График цен** — интерактивный SVG-график по 4 направлениям с лучшим месяцем для покупки
- **💬 Реальные отзывы** — истории экономии от 14 000₽ до 47 000₽ с реальными датами 2026 года

### Технические особенности
- **Live API** — реальные цены подгружаются с серверов Aviasales/Hotellook без посредников
- **Сравнение партнёров** — каждая карточка показывает цены 3-4 систем бронирования, лучшая подсвечена 🏆
- **Mobile-friendly** — адаптивная вёрстка, всё работает на телефонах
- **Автопубликация** — каждый коммит → автосборка → обновление на сайте через 2 минуты

## 🚀 Запуск

### Разработка (открыть на своём компьютере)
```bash
npm install
npm run dev
```
Откроется на **http://localhost:5173**

### Сборка для публикации
```bash
npm run build
node copy-dist.js
```
Результат: **`dist/TravelDeal-website.html`** — весь сайт в одном файле, готов к деплою.

### Просмотр собранной версии
```bash
npm run preview
```

### Запуск backend-сервера (опционально)
Backend с реальными ценами Amadeus API запускается отдельно:
```bash
node server.js
```

## 💰 Монетизация

Сайт подключён к **Travelpayouts** — крупнейшей партнёрской программе для туристических сайтов (Aviasales, Hotellook, Booking и др.).

### Что уже настроено
- ✅ Код Drive встроен в `<head>` сайта (track ID 547188)
- ✅ Все партнёрские ссылки в `src/data/travelData.ts` — личные с `tpk.lu`
- ✅ Раздел «Партнёры» с 23 компаниями-партнёрами
- ✅ Каждый клик = потенциальная комиссия в ваш кабинет

### Сколько зарабатываем
| Тип услуги | Комиссия |
|---|---|
| Бронирование отелей (Hotellook, Ostrovok) | 5-7% от стоимости |
| Авиабилеты (Aviasales) | 1-1.5% от цены |
| Пакетные туры (Травелата, Level) | 3-5% |
| Аренда авто, экскурсии, яхты | 3-8% |
| VIP-залы, хранение багажа | фиксировано за услугу |

**Пример:** турист забронировал через сайт отель в Сочи на 60 000₽ → ваша комиссия ≈ 3000-4000₽.

### Выплаты
- Минимальная сумма вывода: ~500₽
- Способы: карта «Мир», ЮMoney, USDT (TRC20)
- Расчётный период: 1 раз в месяц

### Настройка анкеты
Войдите на **travelpayouts.com** → раздел **«Финансы»** → заполните реквизиты → подтвердите по ссылке из письма → выплаты активируются автоматически.
