## 🚀 Запуск

### Разработка (открыть на своём компьютере)
```bash
npm install
npm run dev
```
Откроется на **http://localhost:5173**

### Сборка для публикации
```bash
npm run build
node copy-dist.js
```
Результат: **`dist/TravelDeal-website.html`** — весь сайт в одном файле, готов к деплою.

### Просмотр собранной версии
```bash
npm run preview
```

### Запуск backend-сервера (опционально)
Backend с реальными ценами Amadeus API запускается отдельно:
```bash
node server.js
```

## 💰 Монетизация

Сайт подключён к **Travelpayouts** — крупнейшей партнёрской программе для туристических сайтов (Aviasales, Hotellook, Booking и др.).

### Что уже настроено
- ✅ Код **Travelpayouts Drive** встроен в `<head>` сайта (track ID **547188**, аккаунт **ID 747557**)
- ✅ Drive подтверждён: уровень монетизации — **максимальный**
- ✅ Все партнёрские ссылки в `src/data/travelData.ts` — личные с `tpk.lu`
- ✅ Каждый клик = потенциальная комиссия в ваш кабинет

### Комиссии по типам услуг
| Тип услуги | Комиссия |
|---|---|
| Бронирование отелей (Hotellook, Ostrovok) | 5-7% от стоимости |
| Авиабилеты (Aviasales) | 1-1.5% от цены |
| Пакетные туры (Травелата, Level, Большая Страна) | 3-5% |
| Аренда авто (Localrent, Qeeq, Economy) | 3-8% |
| Ж/Д билеты (Туту.ру) | фиксировано |
| VIP-залы, хранение багажа, экскурсии | фиксировано за услугу |

**Пример:** турист забронировал через сайт отель в Сочи на 60 000₽ → ваша комиссия ≈ 3 000-4 200₽.
Семья из 4 человек улетела в Дубай за 180 000₽ на билеты → ≈ 1 800-2 700₽.

### Выплаты
- **Минималка:** ~500₽ (или $10 USDT)
- **Способы:** карта «Мир», ЮMoney, USDT (TRC20)
- **Периодичность:** 1 раз в месяц
- **Задержка:** комиссия подтверждается после выезда из отеля / вылета рейса (защита от отмен)

### Настройка выплат
Войдите на **travelpayouts.com** → **Финансы** → **Реквизиты** → заполните анкету (Физлицо, Россия, ФИО, карта Мир или ЮMoney) → подтвердите по ссылке из письма → выплаты активируются.
