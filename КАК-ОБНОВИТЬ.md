# 🔄 Как обновить сайт TravelDeal

## ❗ Проблема
Вы скачали старый архив 295 КБ — там нет новых городов, круизов и перегруппированных партнёров.

## ✅ Решение: 3 способа получить актуальный файл

### Способ 1: GitHub Actions (ЛУЧШИЙ)
1. Зайдите в репозиторий на GitHub
2. Вкладка **Actions** → кликните на **последний успешный запуск** (✅ зелёная галочка)
3. Прокрутите вниз до раздела **Artifacts**
4. Скачайте файл **`gotovyi-sait-dlya-hostinga`**
5. Распакуйте ZIP → внутри **`index.html`** (~350 КБ)
6. Загрузите этот файл в папку `TravelDeal` на хостинге
7. Откройте `sushkova-emma.ru/TravelDeal/` + **Ctrl+Shift+R**

### Способ 2: Прямо из репозитория
1. Зайдите в репозиторий на GitHub
2. Откройте файл `public-built/TravelDeal-website.html`
3. Нажмите кнопку **«Download raw file»** (иконка скачивания)
4. Загрузите файл в папку `TravelDeal` на хостинге

### Способ 3: Скачать большой файл напрямую
1. Зайдите в репозиторий → вкладка **Code**
2. Прокрутите вниз до раздела **Releases** (если есть) — там всегда свежая версия

## 🛠 Если кнопки не нажимаются или файл не скачивается

Попробуйте **в другом браузере** или **в режиме инкогнито** (Ctrl+Shift+N) — там кэша не будет.

## 📋 Проверка что сайт обновился

После обновления на сайте должны появиться:
- В живом поиске отелей 60+ городов (Минводы, Нальчик, Краснодар, Ростов-на-Дону, Махачкала, Камчатка)
- В меню вкладка «**Круизы**»
- В разделе «Партнёры» 6 секций: Авиа, Ж/Д, Отели, Туры, Круизы/Яхты/Авто, Жильё
- Все кнопки бронирования ведут на рабочие страницы (не 404)
