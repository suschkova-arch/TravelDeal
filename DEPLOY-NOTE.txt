ВАЖНО ПО ДЕПЛОЮ

1. Старый файл ~295 КБ = старая standalone-версия.
2. Актуальная React-сборка после последних правок больше: сейчас dist/index.html ~324 КБ.
3. Если на хостинг загружен файл около 295 КБ, значит сайт НЕ обновился.
4. Для актуальной версии загружайте ИМЕННО dist/index.html.
5. Если нужен отдельный файл с понятным названием — используйте TravelDeal-website.html, но только после запуска copy-dist.js, который теперь копирует актуальную React-сборку.

Порядок:
- npm run build
- node copy-dist.js
- загрузить на хостинг файл TravelDeal-website.html ИЛИ dist/index.html
